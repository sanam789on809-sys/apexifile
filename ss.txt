SOFTWARE REQUIREMENTS SPECIFICATION (SRS)
Project Title
Document Management System
1. Introduction
1.1 Purpose
The purpose of this system is to provide a secure web-based document management and workflow platform for an accounting and audit firm. The system will improve collaboration among employees, facilitate secure document sharing between departments and clients, manage task assignments, and provide a centralized repository for all organizational documents.
1.2 Objectives
The system should:
- Provide secure storage for documents.
- Allow departmental collaboration.
- Control document sharing using an approval process.
- Enable communication between staff and clients.
- Improve workflow management through task assignment.
- Provide clients with a secure portal for uploading and downloading files.
- Maintain audit trails for accountability.
- Ensure role-based access control.
2. User Roles
2.1 System Administrator
The administrator shall:
- Manage all users.
- Create departments.
- Create user roles.
- Assign users to departments.
- Activate or deactivate accounts.
- Reset passwords.
- View all files.
- View system logs.
- Configure system settings.
- Generate reports.
- Monitor storage usage.
- Manage client accounts.
- Oversee approval processes.
2.2 Department Head
Department Heads shall be able to:
- Manage departmental files.
- Approve or reject document sharing requests.
- Create folders.
- Upload files.
- Share files within the department.
- Approve sharing with other departments.
- Assign tasks.
- Monitor task progress.
- View completed tasks.
- Generate departmental reports.
2.3 Employee
Employees shall be able to:
- Login securely.
- Upload files.
- Create folders.
- Organize files.
- Share files within their department.
- Request approval for inter-department sharing.
- View files shared with them.
- Download files.
- Search documents.
- Complete assigned tasks.
- Update task progress.
- Receive notifications.
2.4 Client
Clients shall be able to:
- Register.
- Verify email.
- Login.
- Update profile.
- Upload documents.
- Create folders.
- View documents shared by staff.
- Download permitted files.
- Submit support requests.
- View support responses.
- Receive notifications.
